&nbsp;

<span style="font-family: Nunito; font-size: 16px;">In this blog post, we will see how to upgrade vRealize Operations (vROps) from version 6.7.0 to <span class="fileNameHolder">8.0.1. The whole process takes about 30 minutes.</span></span>

<span style="font-size: 16px; font-family: Nunito;"><strong>Bear in mind that, you will need new license keys after updating to the later version (after 8.0 version).<br /> </strong></span>

&nbsp;

<span style="font-family: Nunito; font-size: 16px;"><strong>Note:</strong> You will see a version mismatch in some screenshots.</span>

# <span style="font-family: Nunito; color: #000000; font-size: 32px;">Download the required files</span>

  * <span style="font-family: Nunito; font-size: 16px;">Download the Upgrade Assessment Tool available at <a href="https://my.vmware.com/group/vmware/details?downloadGroup=VROPS-801&productId=940" target="_blank" rel="noopener noreferrer">VMware Downloads</a>.</span>

# <span style="font-family: Nunito; color: #000000; font-size: 32px;"><strong><br /> Upgrade Assessment Tool</strong> (optional but recommended)</span>

<span style="font-family: Nunito; font-size: 16px;"><strong>The first thing to do is to install the Upgrade Assessment Tool in order to know if there are any problems before upgrading.</strong></span>

<span style="font-family: Nunito; font-size: 16px;">Although this is not mandatory it is highly recommended in order to know if there are any issues before running the real upgrade process.</span>

# <span style="font-family: Nunito; color: #000000; font-size: 32px;">Upgrade process</span>

<span style="font-family: Nunito; font-size: 16px;">Before doing anything, <strong>remember to take a snapshot of the Master node</strong> that is going to be upgraded!</span>

&nbsp;

<span style="font-family: Nunito; font-size: 16px;">First, upload the .pak file we downloaded at the beginning of the <em>Virtual Appliance upgrade PAK</em> file (which is way larger than the <em>Upgrade Assessment Tool</em>  PAK file).</span>

<span style="font-family: Nunito; font-size: 16px;">Once downloaded, go to the vROps admin interface (<span style="text-decoration: underline;">https://master-node-FQDN-or-IP-address/admin</span>) and then «Software Updates» :</span><span style="font-family: Nunito; font-size: 16px;"><img loading="lazy" class="alignnone size-full wp-image-1739" src="http://wp.docker.localhost:8000/wp-content/uploads/2020/05/11_Install_SWUpd.png" alt="" width="1792" height="599" srcset="http://wp.docker.localhost:8000/wp-content/uploads/2020/05/11_Install_SWUpd.png 1792w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/11_Install_SWUpd-300x100.png 300w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/11_Install_SWUpd-1024x342.png 1024w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/11_Install_SWUpd-768x257.png 768w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/11_Install_SWUpd-1536x513.png 1536w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/11_Install_SWUpd-1568x524.png 1568w" sizes="(max-width: 1792px) 100vw, 1792px" /></span>

&nbsp;

<span style="font-family: Nunito; font-size: 16px;">And then, upload the new file (which in this case is quite larger compared to the pre-upgrade assessment tool):</span><span style="font-size: 16px;"><img loading="lazy" class="alignnone size-full wp-image-1740" src="http://wp.docker.localhost:8000/wp-content/uploads/2020/05/12.InstallVA_pak.png" alt="" width="1306" height="702" srcset="http://wp.docker.localhost:8000/wp-content/uploads/2020/05/12.InstallVA_pak.png 1306w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/12.InstallVA_pak-300x161.png 300w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/12.InstallVA_pak-1024x550.png 1024w, http://wp.docker.localhost:8000/wp-content/uploads/2020/05/12.InstallVA_pak-768x413.png 768w" sizes="(max-width: 1306px) 100vw, 1306px" /></span>
